# Clariva — AI-Powered Education App

A mobile app concept that helps students learn smarter through 
adaptive quizzes, a personalized AI tutor, and a visual learning 
roadmap. Designed in Figma covering wireframes, high-fidelity UI, 
and a clickable prototype across 7 screens.

## Design System
- Primary color: #534AB7 (purple)
- Font: Inter
- Tool: Figma

## Key UX Decisions
- Splash screen builds brand identity with gradient + logo
- Login kept minimal to reduce signup friction
- Dashboard uses progressive disclosure — AI insight banner 
  surfaces only the most relevant info first
- Quiz uses color-coded feedback (green/red) for immediate 
  status visibility
- Roadmap uses locked/unlocked states to motivate completion
